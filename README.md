# SyncInput

局域网内电脑和手机之间的实时同步输入框。支持文字和图片的双向同步。

## 运行

双击 `syncinput.exe`，手机连同一 WiFi 后扫描二维码或浏览器访问显示的 IP 地址即可连接。
